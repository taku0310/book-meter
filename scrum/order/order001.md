# 依頼事項: Book Meter アプリ開発

## 目的
読書メーターのようなサービスを開発するが、既存サービスよりもデザイン性を高め、スタイリッシュで洗練されたUI/UXを実現する。

## 使用目的
- 読了した本を記録・管理し、後から振り返られるようにする

## 提供媒体
- ブラウザ上で利用できるWebサイト

## 実装したい主な機能 (優先順位: 高)
- 本の検索機能
- 読了済み本の記録機能
- お気に入り登録機能

## デザイン要件
- ミニマリストスタイルを採用
- シンプルで洗練されたUI/UX

## 技術スタック
- TypeScript + React + Next.js

## 期限
- 2026年5月末

## 追加機能
- 不要。シンプルな設計を維持する。

## 備考
- 初期バージョンでは上記主な機能のみを実装する。
